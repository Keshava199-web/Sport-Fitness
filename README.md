## Sport & Fitness Website

A responsive fitness club website built using HTML, CSS, and JavaScript. Designed with modern UI components, smooth navigation, and structured layout principles.

### Features
- Fully responsive design
- Scroll-based navigation
- Interactive hero section
- Class progress indicators
- Newsletter subscription UI
- Modular component structure

### Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Ionicons
- Google Fonts
